# handbook

冒险手册——某只「旅行者·空」的笔记本。

可以在 [GitHub Discussions](https://github.com/chenjunyu19/handbook/discussions) 与我联系、反馈或进行相关讨论。

## 开发与构建

使用最新的 Node.js LTS 版本和 pnpm 包管理工具。

```bash
# 安装依赖
pnpm install

# 启动本地开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
